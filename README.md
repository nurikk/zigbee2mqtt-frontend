# Develop

Install dependencies

```bash
pnpm install
````

Develop using mock data

```bash
npm run start
open http://localhost:3030/
````

Develop using your z2m instance

```bash
Z2M_API_URI="ws://192.168.1.200:8080" npm run start
open http://localhost:3030/
```

# Tests

```bash
npm run test:unit
```

# Build

```bash
npm install
npm run build //compiled files at ./dist
```
